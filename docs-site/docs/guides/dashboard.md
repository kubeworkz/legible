# Dashboard

The dashboard lets you pin charts from your question results to create a visual overview of key metrics.

## Pinning Charts

1. Ask a question in the **Home** section
2. When you get a result with a chart, click the **Pin** icon
3. The chart appears on your dashboard

## Managing the Dashboard

- **Rearrange** charts by dragging them
- **Resize** charts by dragging the edges
- **Remove** a chart by clicking the remove button

## Chart Types

Charts are automatically generated based on the data structure of your query results. The AI selects the most appropriate visualization:

| Data Pattern | Chart Type |
|-------------|------------|
| Single metric | Number card |
| Category + value | Bar chart |
| Time series | Line chart |
| Multiple categories | Grouped bar chart |

## Best Practices

- Pin charts for metrics you check regularly
- Use clear, specific questions to get consistent chart results
- Organize related charts together on the dashboard
